# odin-recipes
TOP Recipes Project

First project on The Odin Project (TOP) that will show basic HTML, git, and GitHub. Enjoyed trying to use CLI as much as possible and trying to get a feel of when to do commits with Git and push them to GitHub.

I think I understood the HTML structure well. I definitely learned that I need to separate the process of creating the content and marking it up (unless it's something really simple). I plan to explore how web developers handle this if they're writing their own content or getting it from a SME or comms/marketing colleague.
