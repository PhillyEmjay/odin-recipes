# odin-recipes

## Description

First project on The Odin Project (TOP) that shows basic HTML, git, and GitHub. Enjoyed using the CLI as much as possible and getting a feel of when to do commits with Git and push them to GitHub.

I think I understood the overall HTML structure well. I definitely learned the need to separate the process of creating the content and marking it up (unless it's something really simple). The project turned into more of a content creation endeavor halfway through and I lost steam. I will explore how web developers handle this if they're writing their own content or getting it from a colleague.

## Revision Notes (Reflection)

Revisiting TOP after a few months of falling off. I was surprised to see that I had not added alt text to a few of the images and that there was no attempt at image attribution. (I'm still unsure of how to proceed here. I don't think any of my images required a credit, but I wanted to put it in to be extra careful.) Decided to put a section in README.md instead of trying to use the figcaption tag. Overall the basic components of the site were okay, but I feel better about it after these revisions.
